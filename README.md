# Movie App Flutter

Movie App is a mobile application for booking movie tickets. Built using Flutter.

## Techologies and Libraries

- Flutter
- Dart
- Dio
- Cloud storage
- Firebase
- Firestore
- BloC
- Go Router
- intl
- Equatable
- Freezed
